# React Web Application

Diese React-Anwendung wurde mit Vite und TypeScript erstellt.

## Installation

```bash
npm install
```

## Entwicklung

Starte den Entwicklungsserver:

```bash
npm run dev
```

## Build

Erstelle eine Produktionsversion:

```bash
npm run build
```

## Technologien

- React
- TypeScript
- Vite
- Node.js
